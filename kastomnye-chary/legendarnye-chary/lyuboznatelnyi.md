# Любознательный

### Применяется к:

* Мечи

### Уровни

#### _1:_&#x20;

* Шанс: 30%
* Перезарядка:  4сек.
* Рассчитывается по формуле ОПЫТ \* (1.0 + 0.25 + ШАНС\_УРОВНЯ%)

#### _3:_&#x20;

### Тип чар

* Убийство мобов

### Описание&#x20;

Шанс увеличить количество ОПЫТА, выпадающих из мобов
