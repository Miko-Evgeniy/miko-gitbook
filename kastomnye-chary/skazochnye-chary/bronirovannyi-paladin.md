# Бронированный Паладин

### Применяется к:

* Броня

### Уровни

#### _1:_&#x20;

* Шанс: 6%
* Перезарядка:  5сек.

#### _2:_

* Шанс: 9%
* Перезарядка:  6сек.&#x20;

#### _3:_&#x20;

* Шанс: 11%
* Перезарядка:  8сек.

#### _4:_

* Шанс: 14%
* Перезарядка:  8сек.&#x20;

### Тип чар

* Защитные

### Описание&#x20;

Снижает урон от вражеских мечей на 2% за уровень

{% hint style="info" %}
Для применения требуется Бронированный IV
{% endhint %}
