# Теневое Уклонение

### Применяется к:

* Броня

### Уровни

#### _1:_&#x20;

* Шанс: 30%
* Перезарядка:  45сек.

### Тип чар

* Защитные

### Описание&#x20;

Шанс увернуться от атаки и получить ускорение, шанс увеличивается при подкрадывании

{% hint style="info" %}
Требуется Уклонение V для применения
{% endhint %}
